
# Application Web - Upload de documents

## 📦 Installation

1. Installer les dépendances :
```bash
cd backend
npm install
```

2. Lancer le serveur :
```bash
npm start
```

3. Ouvrir le navigateur :
```
http://localhost:5000
```

## 📁 Fonctionnement

- Remplir le formulaire HTML
- Les fichiers seront enregistrés dans `/backend/uploads/`
