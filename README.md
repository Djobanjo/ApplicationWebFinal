
# Application Web - Upload de documents

## 📦 Installation

1. Installer les dépendances :
```bash
cd backend
npm install

cd frontend
npm install
```

2. Lancer le serveur :
```bash
cd backend
npm start
```

3. Lancer le frontend : 
```bash
cd frontend
yarn start
```

4. Ouvrir le navigateur :
```
http://localhost:5000
```

## 📁 Fonctionnement

- Remplir le formulaire HTML
- Les fichiers seront enregistrés dans `/backend/uploads/`
